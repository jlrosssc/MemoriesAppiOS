Memories Support

Support information for the Memories iPhone and iPad app, a companion client for a self-hosted Family Memories server.

Getting Started

Memories requires access to a compatible Family Memories server. Open the app, add a saved server profile, sign in with the account created by the server administrator, and use the app menu to open Home, Add Media, Needs Details, Guide, or Admin.

Common Issues

Cannot Connect

Confirm the server URL is correct, starts with https://, and opens in Safari on the same device.

Cannot Sign In

Ask the Memories server administrator to confirm that your account is active and approved. If the site uses HTTP Basic or Digest authentication, confirm the saved username and password in Server Settings.

Upload Problems

Allow Photos, Camera, or Microphone access when prompted. Large videos may take time to upload depending on the network.

Missing Media

Use the server's search filters or Needs Details page. Some media may require permission from the server administrator.

Wrong Server Opens

Open Server Settings, select the desired saved server, and use the Home shortcut to reload that server.
