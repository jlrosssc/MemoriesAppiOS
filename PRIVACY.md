# Memories Privacy Policy

Last updated: July 13, 2026

MemoriesAppiOS is a companion app for a self-hosted Family Memories server. The app connects only to HTTPS server URLs configured by the user.

## Information Stored on Device

The app may store saved server names, server URLs, and usernames on the device. Optional saved passwords are stored in the iOS Keychain. The embedded web view may store server cookies and web session data on the device so the user can stay connected to the selected server.

## Media and Account Data

When a user uploads photos, videos, audio, or documents, that media is sent to the configured Family Memories server. Account credentials, profile data, memory details, and uploaded media are managed by that server and its administrator.

## Camera, Photos, and Microphone

The app may request access to Photos, Camera, or Microphone when the user chooses to upload media through the server's upload forms. The app does not access those resources unless the user starts an upload workflow.

## Analytics and Advertising

The app does not include third-party analytics or advertising SDKs.

## Support

For support, visit https://github.com/jlrosssc/MemoriesAppiOS/issues.
